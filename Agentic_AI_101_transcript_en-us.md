## 二、什麼是 AI 代理 (Agent)？

### 代理的組成比單純 LLM 更複雜：
- **輸入多模態**：文字、圖片、影片、科學模型等
- **記憶**：短期 + 長期記憶（可記住數週前的對話）
- **多模型系統**：語音識別、意圖理解、路由、LLM、圖表理解等模型協同工作
- **子代理**：專門處理特定任務的「小幫手」
- **技能與工具**：
  - 技能：摘要、生成內容、領域專業
  - 工具：命令列、API、瀏覽器 UI、資料庫查詢 (SQL)、非結構化資料（PDF、影片）

### 代理運作流程
```
提示 → 意圖路由器 → 調度器(Orchestrator) → 多個子代理(workers) → 合成結果 → 回覆
```

## 七、安全性：NemoClaw / OpenShell

> 為了解決 **Agent Paradox**：代理越自主（自主性高）→ 需越大的權限 → 越難確保安全

### OpenShell 提供：
- **沙盒 (Sandbox)**：預設拒絕所有權限，由管理員設定策略
- **策略引擎**：從網路層級確保代理不越界
- **隱私路由器**：
  - 敏感任務 → 本地執行、本地模型
  - 需要外部資訊（如網路搜尋）→ 經授權後透過 API 出去（例如 Perplexity API）

✅ NVIDIA 將 OpenShell 全部貢獻給 OpenClaw 開源社群

## 十、如何開始：三步行動建議

### 1️⃣ 直接使用現有工具
- ChatGPT、Perplexity、Claude Code、Claude Work
- 每月 $20 相當於三杯星巴克
- 鼓勵公司 IT 開放使用

### 2️⃣ 解鎖你的資料
- AI 必須能存取你的資料（遵循安全政策）
- 結構化資料：資料庫
- 非結構化資料：透過 RAG 管線處理
- NVIDIA 提供 RAG 藍圖

### 3️⃣ 建立你的個人代理（Build Your Own Claw）
- 安裝 OpenClaw，**慢慢來**：初始設定可對多數權限說「不」
- 逐步開放更多能力
- 你的代理將成為 24/7 的超級幫手
